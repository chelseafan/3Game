Документация по проекту 3GAME


Описание:

3GAME – онлайн игра для 2х человек. Цель игры – за отведенное время набрать больше очков, чем твой соперник. Правила очень просты: перед Вами поле 5х5, где в случайном порядке расположены число. Одним нажатием мышки на свободное поле добавляется цифра, указанная в верхнем левом углу в поле next. Если в результате на поле окажется 3 или более соседних клеток одного номинала, они превращаются в большее число, а сумма уничтоженных клеток добавляется в Ваши очки.


Принцип работы сети:

Проект использует технологию webRTC. Она позволяет обеспечить взаимодействие двух клиентов без специализированного сервера между ними. Клиенты подключаются к облачному серверу, который соединяет их между собой. (Используется библиотека peer.js). Специальный ключ, одинаковый для всех игроков, позволяет установить соединение только с пользователями 3GAME. Если 2 версии игры запущены одновременно, то вскоре они соединятся друг с другом и игра начнется.


Описание файлов:

Проект написан на html и js. Для начала игры необходимо запустить файл index.html

Index.html

Подключается к серверу по удаленным ссылкам, подключая модули game.js и connect.js

Game.js

Основной файл игры, добавляет графику и игровую механику.

Connect.js

Обеспечивает взаимодействие между 2мя клиентами игры. Установка подключения обеспечивается функцией connectRandom из файла randompeer.js. Эта функция стандартна и описана на http://peerjs.com.

Когда подключение установлено, connect.js обеспечивает обмен данными между двумя клиентами.

Сначала, используя блок connection on open, устанавливаются исходные одинаковые поля для двух игроков, начинается отчет таймера

Впоследствии, используется блок connection on data, пересылая значение очков игроков друг другу.
